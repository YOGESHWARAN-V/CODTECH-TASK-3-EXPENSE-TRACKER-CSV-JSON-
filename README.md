# Expense Tracker (CSV)

## Description
This is a simple Expense Tracker application developed using Python. It allows users to add, view, and manage daily expenses using a CSV file.

## Features
- Add new expenses
- View all expenses
- Calculate total expenses
- Save data in CSV file
- Simple Command Line Interface (CLI)

## Technologies Used
- Python
- CSV Module

## How to Run

1. Open the project folder.
2. Run:
   python expense_tracker.py
3. Choose an option from the menu.
